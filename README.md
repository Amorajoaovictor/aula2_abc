## O que e o ABC da programacao?

O ABC da programacao e o conjunto de fundamentos que ajudam uma pessoa a sair do zero e comecar a pensar como programadora.

Antes de criar programas grandes, e importante entender conceitos simples:

- **Entrada**: dados que o programa recebe, como nome, idade, nota ou opcao escolhida.
- **Processamento**: regras que o programa aplica aos dados, como somar notas, calcular media ou comparar valores.
- **Saida**: resultado mostrado para a pessoa usuaria, como uma mensagem, um calculo ou uma classificacao.

Tambem fazem parte desse inicio:

- variaveis;
- operadores;
- condicoes;
- repeticoes;
- funcoes;
- leitura e escrita de dados;
- organizacao do raciocinio antes do codigo.

Os exercicios deste repositorio servem para praticar esses conceitos de forma gradual, com problemas pequenos e objetivos.
